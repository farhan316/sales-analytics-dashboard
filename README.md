# 📊 Sales Analytics Dashboard

> End-to-end sales analytics project using Python (Pandas, Matplotlib, Seaborn) and a Power BI dashboard plan — built on the **Kaggle Superstore Sales dataset**.

---

## 🗂️ Dataset

| Field | Detail |
|---|---|
| **Name** | Sample Superstore Sales |
| **Source** | [Kaggle — Vivek468](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final) |
| **Rows** | ~10,000 orders |
| **Columns** | 21 (dates, products, regions, financials) |
| **Period** | 2014 – 2017 |

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/sales-analytics-dashboard.git
cd sales-analytics-dashboard

# 2. Install dependencies
pip install -r requirements.txt

# 3. Download dataset from Kaggle and place it at:
#    data/superstore_raw.csv

# 4. Run the pipeline
python src/01_data_cleaning.py
python src/02_eda_visualizations.py
python src/03_business_insights.py
```

## 🧹 Data Cleaning Steps

| Step | Action |
|---|---|
| Load | Read CSV with `latin-1` encoding |
| Rename | Standardise columns to `snake_case` |
| Parse Dates | Convert `order_date` & `ship_date` to datetime |
| Deduplication | Drop duplicate `order_id` rows |
| Missing Values | Fill missing postal codes; flag nulls |
| Validation | Remove rows with `quantity ≤ 0` or `sales ≤ 0` |
| Feature Engineering | Add `year`, `month`, `quarter`, `ship_duration`, `profit_margin` |

## 💡 7 Business Insights

1. **Discounts >30% generate negative profits** — Cap promotions at 20%
2. **Tables, Bookcases & Supplies are loss-making** — Review or discontinue
3. **West region leads in revenue AND margin** — Replicate its playbook elsewhere
4. **Q4 drives ~35% of annual revenue** — Prepare inventory by September
5. **Corporate segment has the highest profit margin** — Grow B2B accounts
6. **Same-Day shipping erodes margins by ~8%** — Introduce a surcharge
7. **Revenue grows faster than profit YoY** — Fix discount/cost leakage

## 📊 Power BI Dashboard

The `dashboard/powerbi_design_plan.md` file contains a complete 3-page dashboard blueprint including:

- **Page 1** — Executive KPI Overview (6 KPI cards + map + trend line)
- **Page 2** — Product & Category Deep Dive (scatter, treemap, waterfall)
- **Page 3** — Regional & Time Intelligence (filled map, heatmap, rolling trends)
- Full **DAX measure library** (YoY, YTD, Rolling 3M, QoQ Growth)
- Star-schema **data model** design
- Slicer configuration (Year, Region, Segment, Category)
